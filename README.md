# 🎮 Jogo da Velha em Java

Este projeto foi desenvolvido como parte do programa **ADA – Ser+ Tech**, realizado em parceria com a **Núclea**.

## 📚 Sobre o projeto

O objetivo deste projeto é implementar o clássico **Jogo da Velha** utilizando a linguagem Java, aplicando conceitos fundamentais de programação como:

* Estruturas de repetição (`while`)
* Estruturas condicionais (`if/else`)
* Arrays bidimensionais (matrizes)
* Métodos
* Entrada de dados com `Scanner`

## 🕹️ Como funciona

* O jogo é para **dois jogadores (X e O)**
* Cada jogador escolhe uma posição no tabuleiro 3x3
* O jogo alterna entre os jogadores automaticamente
* O programa valida as jogadas:

    * Não permite posições ocupadas
    * Não permite coordenadas inválidas
* O jogo termina quando:

    * Um jogador vence (linha, coluna ou diagonal)
    * Ou ocorre empate

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/ovitorleal/jogo-da-velha.git
```

2. Abra o projeto no IntelliJ ou outra IDE Java

3. Execute a classe principal:

```bash
JogoDaVelha.java
```
